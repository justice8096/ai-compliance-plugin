# AI Compliance Plugin Skill

This is a Cowork plugin that provides AI compliance capabilities. Use this skill to understand the plugin structure or help the user install/configure it.

## Plugin contents
- **Skills**: Compliance assessment guidance, evidence collection, regulatory lookup
- **Commands**: Quick compliance checks, risk classification, documentation generation
- **Tools**: Integration with compliance-assessment-tools and ai-compliance-extractors

## When to use
- User asks about the AI compliance plugin
- User wants to install or configure compliance tools in Cowork/Claude Code
- User wants to understand what compliance capabilities are available

## Plugin installation
Copy the plugin directory or install from the marketplace. The plugin.json manifest describes all available skills and commands.
